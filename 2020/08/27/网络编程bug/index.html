网络编程bug

发表于 2020-08-27 分类于 Android
本文字数： 1.2k 阅读时长 ≈ 1 分钟

1. 网络请求报错：

  • <uses-permission android:name=”android.permission.INTERNET” / >
  • android:usesCleartextTraffic=”true”
  • 新建xml文件 network_security_config.xml

在application 中添加 android:networkSecurityConfig=”@xml/network_security_config”android:allowBackup=”true

  • <uses-library android:name=”org.apache.http.legacy” android:required=”false” / >

    2、解析json遇到 org.json.JSONException: Value of type java.lang.String cannot be converted to JSONArray

    可能是网络请求报错，也有可能json数据解析时？的值获取错误，检查类的类型和获取值和工具中Object.getString(“weather_id”)

    3、null异常void android.widget.TextView.setText(java.lang.CharSequence)’ on a null object reference

    通常是找不到组件，检查组件的初始化是否出现问题，寻找item前面未加view

    3、使用DrawerLayout出现异常

    布局中允许存放两个直接子控件第一个为主屏幕显示的内容，第二个 为滑动菜单中显示的内容。注意：android:layout_gravity=”start”，为第二个控件必须指定的内容，其中left为从左划出，right为从右划出，start表示根据系统的语言进行判断。

    4、tips

    1、DateSupport已经被取消，改为LitePalSupport

2、ProgressDialog已过时，使用progressbar

3、一般新建project后这里的目录都会默认为Android，首先在把目录结构切换到project上来，这样会使得整个项目结构的构成更为清晰，空的文件夹也不会被隐藏。

4、textAllAaps ：改变所有字母都是大写
