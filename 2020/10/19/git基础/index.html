Git:版本控制工具

1、下载：
  • Mac：https://sourceforge.net/projects/git-osx-installer/
  • Windows：https://git-for-windows.github.io/
2、git基础操作：

Git 所有的操作命令开头都要以 git 开头

git init：初始化 git 仓库

git status：查看git仓库状态

get add ：提交到git仓库（将改动暂时添加到暂存区，临时保存）

git commit -m ‘first commit’：提交的意思，-m 代表是提交信息，第一次提交

git log：查看所有产生的commit记录

git branch：查看当前分支情况

git branch a：新建分支a

git checked a:切换到分支a

git checkout -b a：先建立a分支，并自动切换到a分支

git merge a：合并a分支，在master分支下进行

git branch -d a：删除a分支 -D强制删除

  • ssh-keygen -t rsa：指定rsa算法生成密钥(Linux/Mac 系统 在 ~/.ssh下，win系统在 /c/Documents and Settings/username/.ssh 下)
  • GitHub添加ssh key( 输入验证：ssh -( ssh -T git@github.com )
  • Push&Pull | git clone git@github.com:Three-FISH/sanyu_music.git
